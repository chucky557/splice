# splice
--------------------------------
This is a project I shall be working on to simulate a C2 framework to understand more about red teaming and malware development.
I shall try to use nim in most cases but since I've started falling in love with rust again, I shall also incoporate it. C shall be used when I don't seem to know how to implement something in nim or the feature has not been made available in nim.


#Installation
--------------------------------
1. Clone this github repo using
	```git clone https://github.com/chucky557/splice.git```
2. Change directory to *splice*
	```cd splice```
3. Run make to build the project

#How to use
--------------------------------
Once the project is built, run either *splice* or *main* in the bin folder. This loads the main page of the CLI with a bunner.
To get help on getting started,  type: *help*
